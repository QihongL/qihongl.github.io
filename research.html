Hi, This is Q's research project page. Here're some selected research projects.


1. A recurrent neural network model for human object recognition (2015~)

This is a computational model for the behavioral and neural temporal dynamics of human visual object recognition. Ultra-rapid visual categorization (Wu, et al., 2015) is considered as evidence for a feed-forward view of visual recognition (Serre, Oliva & Poggio, 2007 ). Here, we show that an interactive model can be consistent with this result, and it can also explain many other empirical phenomena, such as the classic basic level advantage.

Link to the 2016 Cognitive Science Society conference poster.

Simulation source code on my Github


2. A reinforcement learning driven neural network for counting (2015~)

Understand how children learn to count, as well as the benefit of social support and enriched feedback, from a computational perspective.

Link to the 2016 Neural Computation and Psychology Workshop poster.

Simulation source code on my Github


3. fMRI decoding with sparse multivoxel pattern analysis (MVPA) (2014~2015)

We investigated how face, place and obejcts represented in the human brain, with a novel method: whole brain iterative logistic LASSO. Different from some conventional ideas (Kanwisher, McDermott, and Chun, 1997), we found that the neural representation of faces is highly distributed across the entire brain.

Link to the 2015 Cognitive Neuroscience Society Conference poster.
