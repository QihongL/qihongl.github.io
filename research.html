Hi, This is Q's research project page. Here're some selected research projects.


1. A recurrent neural network model for human object recognition (2015~)

This is a computational model for the behavioral and neural temporal dynamics of human visual object recognition. Ultra-rapid visual categorization (Wu, et al., 2015) is considered as evidence for a feed-forward view of visual recognition (Serre, Oliva & Poggio, 2007). Here, we show that an interactive model can be consistent with this result, and it can also explain many other empirical phenomena, such as the classic basic level advantage. Currently, we are analyzing an ECoG data set, generously supported by Professor Matt Lambon Ralph, to test a prediction from this model.

2016 Cognitive Science Society conference poster

Simulation code on my Github


2. A reinforcement learning driven neural network for counting (2015~)

Counting skill is a foundation for more sophisticated math concepts. Though it seems to be simple, it takes children several years to learn to do it well. To understand this learning process, we explored how social feedback helps an artificial agent to learn. This modeling framework of this project has been deeply inspired by the Deep Q network (Mnih, et al., 2015). Currently, we are trying to augment this model with an attention mechanism (Mnih et al., 2014).

2016 Neural Computation and Psychology Workshop poster

Simulation code on my Github


3. fMRI decoding with sparse multivoxel pattern analysis (MVPA) (2014~2015)

We investigated how faces, places and objects are represented in the human brain, with a novel method: whole brain iterative logistic Lasso. Different from some conventional ideas (Kanwisher, McDermott, and Chun, 1997), we found that the neural representation of faces is highly distributed across the entire brain.

2015 Cognitive Neuroscience Society Conference poster
