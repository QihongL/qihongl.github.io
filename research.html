Here're some selected research projects!

1. A recurrent neural network for human object recognition

This is a computational model for the behavioral and neural temporal dynamics of human visual object recognition. Ultra-rapid visual categorization (Wu, et al., 2015) is considered as evidence for a feed-forward view of visual recognition (Serre, Oliva & Poggio, 2007). Here, we show that an interactive model can be consistent with this result, and it can also explain many other empirical phenomena, such as the classic basic level advantage. Currently, we are analyzing an ECoG data set, generously supported by Professor Matt Lambon Ralph, to test a prediction from this model.

2016 Cognitive Science Society conference poster, Simulation code





2. A reinforcement learning network for "counting"

Counting skill is a foundation for more sophisticated math concepts. Though it seems to be simple, it takes children several years to learn to do it well. To understand this learning process, we explored how social feedback helps an artificial agent to learn a counting-related task. This modeling framework of this project has been deeply inspired by the Deep Q network (Mnih, et al., 2015). Currently, we are trying to augment this model with hard attention (Mnih et al., 2014).

2016 Neural Computation and Psychology Workshop poster, Simulation code





3. Discover distributed representation via fMRI decoding

We investigated how faces, places and objects are represented in the human brain, with a novel multi-voxel pattern analysis (MVPA) method: whole-brain iterative logistic Lasso. Different from some conventional findings (Kanwisher, McDermott, and Chun, 1997), we found that the neural representation of faces is highly distributed across the entire brain.

2015 Cognitive Neuroscience Society Conference poster
